# Meta 10-K Financial Analysis 📊

This project analyzes the financial performance of Meta Platforms, Inc. using data from its official 10-K filing.

## 📌 Overview
The goal of this project is to visualize key financial metrics such as revenue and operating cash flow over a 3-year period (2023–2025) using Python.

## 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib
- Google Colab

## 📊 Data Source
- Meta Platforms, Inc. 10-K (SEC Filing)

## 📈 Key Metrics Analyzed
- Total Revenue
- Operating Cash Flow

## 🔍 Key Insights
- Meta’s revenue shows strong and consistent growth from 2023 to 2025.
- Operating cash flow also increased significantly, indicating strong cash generation.
- The company demonstrates improved financial performance and operational efficiency.

## 📷 Visualizations
The project includes:
- Revenue trend chart
- Operating cash flow chart
- Combined comparison chart

## 🚀 How to Run
1. Open the notebook in Google Colab
2. Run all cells
3. View generated charts

## 💡 Conclusion
This analysis highlights Meta’s financial growth and its ability to generate strong cash flows, driven primarily by its advertising business.
